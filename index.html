How to Remove Vocals from a Song

Follow these steps to create karaoke tracks or isolate vocals using Marin AI's vocal remover:

  1. Select Audio Source: Choose whether to input a YouTube or SoundCloud link or upload an MP3/WAV file. Use the "Audio Source" dropdown to make your selection.
  2. Provide Audio Input: If selecting a link, paste the valid YouTube or SoundCloud URL in the "Audio URL" field. For file uploads, click the "Upload Audio File" field to select an MP3 or WAV file from your device.
  3. Choose Number of Stems: Select the number of stems (2, 4, or 5) you want to separate from the audio using the "Number of Stems" dropdown. Stems can include vocals, drums, bass, piano, and other instruments, depending on the selection.
  4. Select Output Preference: Choose between "Process with Mixer" to adjust the separated stems interactively or "Direct Download ZIP" to receive all stems as a downloadable ZIP file.
  5. Start Processing: Click the "PROCESS" button to initiate vocal removal. A loading screen will appear with the message "Please wait, our AI models are processing..." and a progress video. Processing typically takes about one minute.
  6. Download or Adjust Output: Once processing is complete, you will either be directed to the mixer interface (if selected) to fine-tune the stems or prompted to download a ZIP file containing the separated stems.
  7. Review and Use: Use the isolated stems for karaoke, remixing, or music production. Ensure the output meets your needs, and reprocess with different settings if necessary.

Note: For best results, use high-quality audio sources. Ensure you have the necessary permissions to process and use the audio for your intended purpose.
